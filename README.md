# MY_Akari_Program
Akari (LIGHTS UP) logic puzzle solver implemented with the backtracking algorithm, with partial code pending debugging, including the project report and source code.

## 1. 项目介绍
这是一个用于解决Akari逻辑谜题的C++程序，实现了谜题的自动化求解功能

## 2. Akari谜题规则
本程序遵循的Akari谜题规则：
1.在图中设置灯泡，使得所有白色格子都被照亮。灯泡只能放在白格里。
2.每个灯泡照亮的范围为以灯泡为中心的十字形状，每端照亮至图的边缘或黑格。
3.黑格里的数字代表上下左右四格里有多少个灯泡。
4.不容许任一灯泡照到其他灯泡。
可假定给出的谜题都有唯一解。

## 3. 解题思路
程序通过五步流程完成谜题求解，核心逻辑基于回溯法实现：
1. **数据初始化**：
   将输入字符串转换为二维矩阵，同时初始化辅助数组（记录黑格位置、灯泡放置状态、光照覆盖状态等）。
2. **约束条件设置**：
   通过函数实现两个核心约束：
   - 黑格数字对应的相邻灯泡数量需匹配；
   - 灯泡之间不能互相照射。
3. **预判定灯泡位置**：
   先确定“必须放灯泡”“绝对不能放灯泡”的位置，减少回溯复杂度。
4. **回溯求解**：
   通过回溯函数遍历所有可能的灯泡放置组合，结合约束条件筛选合法解。
5. **结果输出**：
   回溯得到正确解后，按照题目要求格式输出（用“L”标记灯泡位置）。

## 4. 使用说明
1. 运行程序后，输入两个整数（代表谜题的行数、列数）；
2. 输入谜题内容（格式示例：`..1X...X.1..X....0X.....1.2.X....1....2..1X..`，其中`.`为空格、`1``2`为带数字黑格、`X`为普通黑格）；
3. 程序输出结果（格式与输入一致，用`L`标记灯泡位置）。
